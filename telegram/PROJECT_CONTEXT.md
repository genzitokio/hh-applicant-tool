# Контекст проекта: Сбор Telegram IT-рекрутеров

> Документ описывает общий контекст задачи и историю исследования

## Изначальная задача

Пользователь хотел найти контакты IT-рекрутеров. Первоначально пробовали Хабр Карьера, но:

### Хабр Карьера — почему не подошёл

1. **Контакты заблокированы** — при клике на "Открыть контакты":
   > "Контакты недоступны. Открыть контакты могут кураторы компаний с оплаченным доступом к базе."

2. **Поиск ограничен** — показывает только 7 профилей, остальные 605+ требуют авторизации компании:
   > "Поиск ограничен. Полный поиск по базе соискателей доступен кураторам компаний с размещёнными вакансиями или оплаченным доступом к базе."

3. **API закрыт** — `GET /api/frontend_v1/resumes` возвращает `{"list":[],...}` — пустой массив без доступа компании

4. **Данные не в DOM** — контакты реально не передаются клиенту, это не CSS-маскировка

### Вывод по Хабр Карьера

Хабр грамотно защитил данные на уровне сервера. Без платного корпоративного доступа получить контакты невозможно.

---

## Смена вектора → Telegram OSINT

Решили сфокусироваться **только на Telegram** как источнике контактов IT-рекрутеров.

### Почему Telegram

1. **Прямой доступ к username** — участники чатов видны
2. **Публичные данные** — участие в публичных группах = публичная информация
3. **Большие HR-сообщества** — чаты с 8-20k участников
4. **Bio с контактами** — многие рекрутеры указывают TG/email в bio
5. **API доступен** — Telethon/Pyrogram позволяют парсить

### Цель

> **Собрать максимум Telegram-контактов IT-рекрутеров, используя OSINT-инструменты (бесплатные и open-source)**

---

## Что было исследовано

### 1. Источники данных
- Telegram HR-чаты (приоритет)
- Каналы IT-рекрутеров
- TGStat/Telemetr каталоги
- Поисковые системы (Lyzem и др.)

### 2. Методы парсинга
- Telethon MTProto API
- Готовые GitHub scrapers
- Лимиты API (FloodWait, aggressive mode)

### 3. OSINT инструменты
- Боты для поиска групп (@TgScanRobot)
- Боты для инфо о юзерах (@IDBot)
- Пробив-боты (Глаз Бога и др.)

### 4. Поисковые системы
- TGStat (2.6M+ каналов, API)
- Lyzem (глубокий поиск)
- Telemetr (аналитика)

## Связь с AutoKus

Проект AutoKus — HR-бот для автоматического общения с рекрутерами. Собранная база контактов может использоваться для:

1. **Проактивного outreach** — написать рекрутерам первым
2. **Whitelist expansion** — расширить список HR для автоответов
3. **Market research** — анализ активности рекрутеров
