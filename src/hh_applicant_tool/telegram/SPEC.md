1# AutoKus - HR Bot Specification

## Пользователь
- **Роль:** Senior Frontend Developer
- **Стек:** React + TypeScript, Next.js/Nuxt
- **Формат работы:** Любой (удалёнка, офис, гибрид)
- **География:** Только РФ
- **ЗП:** По ситуации (гибко)
- **Красные флаги:** Нет ограничений
- **Тестовые задания:** Не берёт вообще

## Источники HR
- Готовые контакты (whitelist)
- HR пишут сами
- Поиск в группах — позже
- Объём: нужно продумать лимиты (антиспам)
- Старт с нуля, активных переписок нет

## Сценарии общения
- **Первый контакт:** HR → бот или бот → HR
- **Переговоры о ЗП:** Гибко по ситуации
- **Созвоны:** Любое время, без ограничений
- **Тестовые:** Отказ
- **Стиль:** Формальный + подстраиваться под собеседника
- **Язык:** Только русский

## Тайминги
- **Задержка ответа:** 2-5 минут (мин), 30 минут (макс)
- **Рабочие часы:** 9:00-21:00
- **Выходные:** Да, как в будни

## Контроль и безопасность
- **Подтверждение действий:** Никакое (полный автомат)
- **Эскалация:** Никогда (бот справляется сам)
- **Запреты:** Нет ограничений (может всё)

## Классификация
- **Метод:** Whitelist контактов
- Отвечать только на сообщения от HR из whitelist

## Мониторинг
- **Алерты:** В Telegram на @genziii
- **События:**
  - Важное (оффер, созвон)
  - Ошибки
  - Сводка за день

## Техническое
- **Управление:** Через Claude Code (skill или просьба)
- **LLM:** Claude Code, Gemini, GLM 4.7
- **Хранение:** JSON/YAML файл (whitelist, настройки)
- **История чата:** Из Telegram API напрямую
- **Среда:** Локально на ПК (Arch Linux)
- **Запуск:** Через Claude Code

## Функции MVP
- Автоответы на сообщения HR
- Человеческие задержки (2-30 мин)
- Рабочие часы (9-21)
- Whitelist контактов
- Формальный стиль + адаптация
- Переговоры о ЗП
- Назначение созвонов
- Отказ от тестовых
- Алерты в Telegram
- Аналитика/статистика
- Шаблоны ответов на FAQ
